# UI 设计规范 - 历史粘贴板

## 设计原则

- **简洁**：减少视觉噪音，只展示必要信息
- **直观**：操作符合直觉，不需要学习成本
- **高效**：最小化操作步骤，一键完成核心任务

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#f0f6fc` | 淡蓝灰，柔和护眼 |
| 卡片背景 | `#ffffff` | 纯白，与背景形成层次 |
| 主色调 | `#4a90d9` | 淡蓝，按钮/强调/选中态 |
| 主色调悬停 | `#3a7bc8` | 深一度的蓝，hover 状态 |
| 主文字 | `#333333` | 深灰，确保可读性 |
| 次要文字 | `#999999` | 浅灰，时间戳等辅助信息 |
| 分隔线 | `#e8eef4` | 极淡蓝灰 |
| 危险色 | `#e74c3c` | 红色，删除按钮 |
| 成功色 | `#27ae60` | 绿色，成功提示 |
| 置顶标识 | `#4a90d9` | 淡蓝，与主色调一致 |

## 字体

### 文本卡片

- 内容预览：最多显示 2 行，超出用省略号
- 行高：1.5
- 文字颜色：#333

### 图片卡片

- 缩略图高度：最大 120px
- 缩略图宽度：填充卡片宽度
- 图片以 object-fit: cover 显示

### 卡片操作按钮

- 置顶按钮：卡片右上角，图钉图标，默认隐藏悬停显示
- 删除按钮：卡片右上角，垃圾桶图标，默认隐藏悬停显示
- 置顶态：卡片左侧有淡蓝色竖条标识（3px 宽）
- 颜色：默认 #ccc，悬停变主色/危险色

## 设置面板

- 从右侧滑入的抽屉面板
- 宽度：280px
- 背景：#ffffff
- 保留天数使用分段控件（Segmented Control）
- 三个选项：1天 / 3天 / 5天
- 选中的选项以淡蓝背景 + 白色文字突出

## 空状态

- 居中显示
- 图标：空剪贴板插图（SVG）
- 文字："暂无剪贴记录"
- 副文字："试试复制一些内容吧"
- 文字颜色：#999

## 交互行为

| 操作 | 行为 |
|------|------|
| Ctrl+Shift+V | 切换面板显示/隐藏 |
| 点击文本卡片 | 复制内容到剪贴板，面板隐藏 |
| 点击图片卡片 | 复制图片到剪贴板，面板隐藏 |
| 悬停卡片 | 显示置顶和删除按钮 |
| 点击置顶按钮 | 切换置顶状态，置顶项移到顶部 |
| 点击删除按钮 | 软删除，卡片从列表移除，显示 Toast |
| 搜索输入 | 实时过滤列表（300ms 防抖） |
| Esc 键 | 隐藏面板 |
| 点击面板外部 | 隐藏面板 |
