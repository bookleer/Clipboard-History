# 需求文档 - 历史粘贴板

## 项目概述

一款运行在 Windows 上的剪贴板历史管理软件，持续监听并记录用户复制的文本和图片，按时间降序展示，支持搜索、置顶、删除，可设定保留天数。

## 目标用户

需要在日常工作中频繁使用复制粘贴的 Windows 用户，希望能够回溯历史复制内容。

## 功能清单

### 核心功能

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F1 | 剪贴板监听 | 后台自动记录用户复制的一切文本和图片 | P0 |
| F2 | 历史面板 | 以时间降序卡片形式展示所有复制记录 | P0 |
| F3 | 搜索功能 | 在历史记录中通过关键词搜索 | P1 |
| F4 | 置顶功能 | 将重要记录固定在列表顶部 | P1 |
| F5 | 删除功能 | 删除不需要的记录（软删除） | P1 |
| F6 | 保留天数设置 | 可设定 1天/3天/5天，过期自动清理 | P1 |
| F7 | 点击复制 | 点击历史卡片即可将内容重新复制到剪贴板 | P0 |

### 辅助功能

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F8 | 全局快捷键 | Ctrl+Shift+V 呼出/隐藏历史面板 | P0 |
| F9 | 系统托盘 | 托盘图标常驻，右键菜单操作 | P1 |
| F10 | 开机自启 | 开机后自动在后台运行 | P2 |
| F11 | 内容去重 | 连续复制相同内容不会重复记录 | P1 |

## 验收标准

1. 复制文本后，按 Ctrl+Shift+V 打开面板，能看到刚复制的文本
2. 复制图片（如截图）后，面板中能看到图片缩略图
3. 点击历史文本卡片后，可以粘贴到其他应用
4. 点击历史图片卡片后，可以粘贴到其他应用
5. 搜索关键词能正确过滤列表
6. 置顶的项目始终排列在最前面
7. 删除的项目从列表中消失
8. 过期记录在设定天数后被自动清理
9. 重启电脑后软件自动启动并在后台运行

## 非功能需求

- 性能：剪贴板监听间隔 500ms，UI 响应流畅
- 安全：数据存储在本地，不上传网络
- 兼容性：Windows 10/11
- 安装包大小：尽量控制在 100MB 以内
